# Multi Agent Orchestrator
## Project Description
- multi agent system where multiple agents run in parellel doing their tasks
- user can manage agents (add, edit, remove) and monitor what they're doing
- each agent can be assigned to developers, product manager, designer, project manager (leader of the project)
- user can add roles with role name and description
- all of agent should communicate in slack (preferably different account for each agent)
- pr must be created when pushing commits
- they can create documents in google docs or similar platforms
- they run in background.

## Tech Stack
- Node.js
